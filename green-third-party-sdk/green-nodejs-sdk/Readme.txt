运行提示：
#1 确保Node.js环境安装成功
#2 imageSyncScanSample.js中accessKeyId accessKeySecret 替换为自己阿里云账户AK
#3 uuid模块安装，参考命令：npm install uuid  详见：https://www.npmjs.com/package/uuid
#4 green-nodejs-invoker.js和imageSyncScanSample.js确保在同一目录下，或者修改imageSyncScanSample.js使之定位到green-nodejs-invoker.js
#5 编写自己的业务处理函数execute，参数为绿网API返回结果；终端运行imageSyncScanSample.js即可
